# Hackaton NS

A hackathon project.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/andreferd/hackaton_NS.git
   cd hackaton_NS
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

### Running the Project

```bash
npm start
```

Or for development mode:

```bash
npm run dev
```
